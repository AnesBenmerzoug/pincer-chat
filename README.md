## Getting Started

- Download and install [Ollama](https://ollama.com/download)

- Start ollama server:

  ```shell
  ollama serve
  ```

- Start the application:

  ```shell
  cargo install --git https://github.com/AnesBenmerzoug/pincer-chat.git
  ```
